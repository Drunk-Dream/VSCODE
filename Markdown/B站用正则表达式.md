# 正则表达式

```regex
((\d{1,2}|[一二三四五六七八九十两])(分钟?|小时|秒|s|S|min|播放))|热乎的?|第[一二三]|刚刚|前排|沙发|承包|来了|烫[手嘴]
测试词：1分钟，10分钟，一小时，十小时，热乎的，热乎，第一，第二，第三，沙发，承包空屏！，来了，高调承包，烫手，烫嘴，1播放
```
